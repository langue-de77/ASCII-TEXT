# ASCII TEXT

テキストをASCIIアートに変換するWebアプリです．

![ASCII TEXT](./img/favicon.ico)

## 概要

入力したテキストを，`0`と`1`（または任意の2文字）で構成されたASCIIアートに変換します．横書き・縦書きの両モードに対応しています．

## 機能

- テキストをASCIIアートに変換
- 背景文字・主文字を自由にカスタマイズ可能
- 横書き・縦書きの切り替え
- 変換結果をテキストエリアに出力

## 対応文字

`A`〜`Z`の英大文字26文字，`0`〜`9`の数字10文字，およびスペースに対応しています．

## ファイル構成

```
プロジェクト/
├── index.html       # メインのHTMLファイル
├── app.js           # 変換ロジック
├── style.css        # スタイルシート
├── img/
│   └── favicon.ico
└── data/            # 各文字のASCIIアートデータ
    ├── A.at
    ├── B.at
    ├── ...
    ├── Z.at
    ├── 0.at
    ├── ...
    ├── 9.at
    └── space.at
```

## 使い方

1. このリポジトリをクローンする

```bash
git clone https://github.com/your-username/ascii-text.git
cd ascii-text
```

2. ローカルサーバーを起動する

`fetch`を使用しているため，ファイルを直接ブラウザで開くと動作しません．VSCodeの**Live Server**拡張などを使ってください．

3. ブラウザで `index.html` を開く

4. テキストエリアに変換したい文字列を入力し，「変換」ボタンを押す

## カスタマイズ

「背景文字」「主文字」の入力欄に任意の文字を入力することで，ASCIIアートを構成する文字を変更できます．

| 項目     | デフォルト | 説明                            |
| -------- | ---------- | ------------------------------- |
| 背景文字 | `0`        | ASCIIアートの背景部分に使う文字 |
| 主文字   | `1`        | ASCIIアートの文字部分に使う文字 |

例として `■` と `□` を設定すると，ブロック文字によるASCIIアートになります．

## `.at`ファイルのフォーマット

`data/`フォルダ内の各`.at`ファイルは，`0`と`1`で構成されたグリッドデータです．

```
000000000000001100000000000000
000000000000111111000000000000
000000000011110011110000000000
000000001111111111111100000000
000000001111000000111100000000
000000111100000000001111000000
```

- `0` → 背景文字に置換される
- `1` → 主文字に置換される

## 技術スタック

- HTML / CSS / JavaScript（純粋なフロントエンドのみ，フレームワーク不使用）
- `fetch` API によるファイル読み込み
- キャッシュによる重複読み込みの抑制
